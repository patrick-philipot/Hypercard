Fichiers (lecture et écriture)

Les fichiers créés sont directement utilisables par un éditeur de texte. Dans RESEDIT, creator et type sont respectivement ttxt et TEXT. 

      on mouseUp
      --générer un fichier avec ce script
      put "script.txt" into newfile
      open file newfile
      --write bg field "about" to file newfile
      write the script of me to  newfile
      close  newfile
      
      --relire le fichier
      put "script.txt" into filename
      open  filename
      
      if the result is not empty then
        answer "Impossible d'ouvrir le fichier" && filename
        exit mouseUp
      end if
      
      -- lit tout le fichier
      read from  filename until end
      put it into cd field "liste"
      close  filename
    end mouseUp
