Ask and Answer

Ask and Answer are HyperCard commands that produce special dialog boxes. They are useful for giving the feel of interactivity in a stack

Answer produces dialog boxes that require users to select from up to 3 buttons

Ask produces a dilaog box requiring the user to type a response
