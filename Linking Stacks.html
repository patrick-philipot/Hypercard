Using Stacks

La commande "start using stack stack_name" permet d'utiliser les scripts de la stack désignée, notamment le script de ses handlers. La commande "stop using stack stack_name" annule cette action. La commande "stacksInUse" retourne la liste des stacks chargées par "start using stack ..."

  
    --EXEMPLE DE CODE
    put the stacksInUse into usedStacks
    -- retourne par exemple
    -- "HD80FR:HyperCard 2.4:HyperCard Stacks:Audio Help"
    if usedStacks contains "Audio Help" then
        -- inutile de donner le chemin complet
        stop using stack "Audio Help"
    end if
